Bienvenue sur Urban Marginal, jeu de tir multijoueurs en 2D.


// -----------------------------------------
//
// 		LANCER UNE PARTIE
//
// -----------------------------------------

Le jeu se lance en cliquant sur le fichier "Urban Marginal.jar".

Pour lancer une partie un joueur doit lancer un serveur en renseignant son adresse ip,
puis en cliquant sur Start. 
Ensuite chaque joueur se connecte au serveur après avoir renseigné l'adresse ip de celui-ci
puis en cliquant sur le bouton Connect.
